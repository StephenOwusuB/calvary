Objectives

In this project, we will set up a secure Active Directory home lab environment using Oracle VirtualBox. This hands-on lab aims to enhance our understanding of securing an organization's Active Directory environment by implementing robust security measures, best practices, and automated scripts. Configuring and managing this lab significantly contributed to my knowledge of maintaining the security and integrity of Active Directory systems.

    Prerequisites

    To set up the environment for the Securing Active Directory project, you will need the following components:

    • Windows Server on VirtualBox
      • Download and install VirtualBox.
      • Create a new virtual machine and install a Windows Server operating system (e.g., Windows Server 2016 or later).
      • Configure the virtual machine with sufficient resources (CPU, RAM, disk space) to support Active Directory services.
    • Custom Domain from Namecheap
      • Purchase a custom domain from Namecheap.
      • Configure DNS settings to point to your Windows Server running on VirtualBox.
      • Set up the domain to be used within your Active Directory environment.
    • PowerShell Modules
      • Ensure you have PowerShell installed on your Windows Server.
      • Install the necessary PowerShell modules for managing Active Directory. Open PowerShell and run the following commands: 
        Install-Module -Name ActiveDirectory
Install-Module -Name DNSClient
Install-Module -Name PSWindowsUpdate

    Skills Learned

    • Active Directory Management: Learn how to set up, configure, and manage an Active Directory environment effectively.
    • Security Best Practices: Understand and implement best practices for securing Active Directory and mitigating potential vulnerabilities.
    • Automation Scripting: Gain proficiency in writing and using automated scripts to streamline AD management tasks.
    • User and Group Management: Learn how to manage user accounts and groups within Active Directory securely.
    • Backup and Recovery: Develop skills in implementing automated backup solutions to ensure data integrity and availability.

    Tools Used

    • PowerShell: For writing automated scripts to manage and secure Active Directory.
    • Active Directory Module: To perform various administrative tasks related to Active Directory.
    • Git: For version control and collaboration.
    • Visual Studio Code: As the primary code editor for writing and managing scripts.
    • Markdown: For creating detailed documentation and guides.
